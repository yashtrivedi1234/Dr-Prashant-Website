# Dr. Prashant - Website

This project is the official website for Dr. Prashant, a leading Homeopathy Doctor & Healthcare Advocate.

## Project info

The website features a modern, responsive design built with React and Tailwind CSS, showcasing Dr. Prashant's services, about section, and contact information.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Development

Follow these steps to run the project locally:

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm run dev
   ```

3. Build for production:
   ```sh
   npm run build
   ```
